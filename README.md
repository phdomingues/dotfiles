# dotfiles
My personal dotfiles for Arch


# Installation
## GTK-Theme

```
yay -S adapta-gtk-theme
```

## Qt5 theme
```
sudo pacman -S kvantum-qt5
```

## Install flameshot
# as per 06-2025, flameshot does not work well with wayland, but flameshot-git set to use grim adapter works flawlessly
```
yay -S flameshot-git
```
